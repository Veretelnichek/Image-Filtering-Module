## Медианный фильтр

**Функция:** `median_filtrs(source_name, f)`

Применяет медианную фильтрацию по вертикали с окном в 3 пикселя. Для каждого цветового канала (R, G, B) независимо вычисляется медиана трёх соседних по вертикали значений, и полученное значение записывается в текущий пиксель. Процедура повторяется `f` раз.

### Параметры

| Параметр      | Тип   | Описание                                                                               |
| ------------- | ----- | -------------------------------------------------------------------------------------- |
| `source_name` | `str` | Путь к входному изображению (JPEG, PNG, BMP и др.)                                     |
| `f`           | `int` | Количество повторных применений фильтра (чем больше значение, тем сильнее сглаживание) |

### Алгоритм

1. Изображение загружается как трёхмерный массив `(высота, ширина, каналы)`.
2. Для каждого прохода:

   * Берутся три среза: строки `[0:h-2]`, `[1:h-1]` и `[2:h]`, где `h` — высота изображения.
   * Для каждой позиции в первых `h-2` строках вычисляется медиана трёх соответствующих пикселей по вертикали.
   * Результат записывается в соответствующие строки выходного изображения.
   * Две последние строки остаются без изменений.
3. Процесс повторяется `f` раз, причём каждый следующий проход выполняется над результатом предыдущего.

### Пример использования

```python
from filtr import median_filtrs

median_filtrs("input.jpg", 2)   # применить фильтр дважды
```

Результат сохраняется в текущей директории под именем:

```text
median_filtrs.jpg
```

## Свёрточный (матричный) фильтр

**Функция:** `filter_matrixSvertki(source_name, result_name, div)`

Выполняет линейную свёртку изображения с фиксированным ядром размером `3×3`:

```text
ядро = [[0.5, 0.75, 0.5],
        [0.75, 1.0,  0.75],
        [0.5,  0.75, 0.5]]
```

Данное ядро представляет собой сглаживающий (низкочастотный) фильтр, близкий по эффекту к гауссовскому размытию, но с нестандартными весовыми коэффициентами. Обработка применяется ко всем внутренним пикселям изображения, за исключением краевых.

### Параметры

| Параметр      | Тип   | Описание                                                                                                                      |
| ------------- | ----- | ----------------------------------------------------------------------------------------------------------------------------- |
| `source_name` | `str` | Путь к входному изображению                                                                                                   |
| `result_name` | `str` | Имя файла для сохранения результата (например, `"output.jpg"`)                                                                |
| `div`         | `int` | Делитель для нормализации суммы свёртки. Обычно используется значение `6`, позволяющее сохранить исходную яркость изображения |

При использовании `div = 1` изображение станет значительно ярче.

### Алгоритм

1. Изображение загружается в массив NumPy.
2. Создаётся копия изображения, в которой:

   * первые две строки и первые два столбца копируются без изменений;
   * последняя строка и последний столбец обнуляются для предотвращения артефактов на границах;
   * все внутренние пиксели пересчитываются с использованием операции свёртки.
3. Для каждого внутреннего пикселя вычисляется:

```text
результат = (сумма произведений пикселей окна на соответствующие веса ядра) / div
```

4. Полученные значения ограничиваются диапазоном `[0, 255]` и сохраняются как целые числа.

### Пример использования

```python
from filtr import filter_matrixSvertki

filter_matrixSvertki("input.jpg", "blurred.jpg", 6)
```

## Установка и использование

### Требования

* Python 3.7 или выше
* Библиотеки:

  * `numpy`
  * `Pillow`

### Установка зависимостей

Создайте файл `requirements.txt` со следующим содержимым:

```text
numpy
Pillow
```

Затем выполните:

```bash
pip install -r requirements.txt
```

## Запуск

Импортируйте необходимую функцию из модуля `filtr` и вызовите её с нужными параметрами:

```python
from filtr import median_filtrs, filter_matrixSvertki

# Медианный фильтр
median_filtrs("photo.jpg", 3)

# Свёрточный фильтр
filter_matrixSvertki("photo.jpg", "result.jpg", 6)
```
