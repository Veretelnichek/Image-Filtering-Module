# Медианный фильтр изображений

Данный проект реализует **медианную фильтрацию** изображений по вертикали с окном 3 пикселя. Алгоритм применяется к каждому цветовому каналу (R, G, B) независимо и может повторяться несколько раз для усиления эффекта сглаживания. Фильтр эффективно удаляет импульсные шумы («соль и перец»), сохраняя при этом резкость границ.

## Особенности
- Использование векторизованных операций **NumPy** для высокой производительности.
- Автоматическая обработка цветных изображений (RGB).
- Возможность многократного применения (параметр `f`).
- Сохранение результата в формате JPEG.

## Требования
- Python 3.7 или выше
- Библиотеки: `numpy`, `Pillow`
