## Медианный фильтр

**Функция:** `median_filtrs(source_name, f)`

Применяет медианную фильтрацию по вертикали с окном в 3 пикселя. Для каждого цветового канала (R, G, B) независимо вычисляется медиана трёх соседних по вертикали значений, и полученное значение записывается в текущий пиксель. Процедура повторяется `f` раз.

### Параметры

| Параметр      | Тип   | Описание                                                                               |
| ------------- | ----- | -------------------------------------------------------------------------------------- |
| `source_name` | `str` | Путь к входному изображению (JPEG, PNG, BMP и др.)                                     |
| `f`           | `int` | Количество повторных применений фильтра (чем больше значение, тем сильнее сглаживание) |

### Алгоритм

1. Изображение загружается как трёхмерный массив `(высота, ширина, каналы)`.
2. Для каждого прохода:

   * Берутся три среза: строки `[0:h-2]`, `[1:h-1]` и `[2:h]`, где `h` — высота изображения.
   * Для каждой позиции в первых `h-2` строках вычисляется медиана трёх соответствующих пикселей по вертикали.
   * Результат записывается в соответствующие строки выходного изображения.
   * Две последние строки остаются без изменений.
3. Процесс повторяется `f` раз, причём каждый следующий проход выполняется над результатом предыдущего.

### Пример использования

```python
from filtr import median_filtrs

median_filtrs("input.jpg", 2)
```

Результат сохраняется в текущей директории под именем:

```text
median_filtrs.jpg
```

## Свёрточный (матричный) фильтр

**Функция:** `filter_matrixSvertki(source_name, result_name, div)`

Выполняет линейную свёртку изображения с фиксированным ядром размером `3×3`:

```text
ядро = [[0.5, 0.75, 0.5],
        [0.75, 1.0,  0.75],
        [0.5,  0.75, 0.5]]
```

Данное ядро представляет собой сглаживающий (низкочастотный) фильтр, близкий по эффекту к гауссовскому размытию, но с нестандартными весовыми коэффициентами. Обработка применяется ко всем внутренним пикселям изображения, за исключением краевых.

### Параметры

| Параметр      | Тип   | Описание                                                                  |
| ------------- | ----- | ------------------------------------------------------------------------- |
| `source_name` | `str` | Путь к входному изображению                                               |
| `result_name` | `str` | Имя файла для сохранения результата                                       |
| `div`         | `int` | Делитель для нормализации суммы свёртки. Обычно используется значение `6` |

При использовании `div = 1` изображение станет значительно ярче.

### Алгоритм

1. Изображение загружается в массив NumPy.
2. Создаётся копия изображения, в которой:

   * первые две строки и первые два столбца копируются без изменений;
   * последняя строка и последний столбец обнуляются для предотвращения артефактов на границах;
   * все внутренние пиксели пересчитываются с использованием операции свёртки.
3. Для каждого внутреннего пикселя вычисляется:

```text
результат = (сумма произведений пикселей окна на соответствующие веса ядра) / div
```

4. Полученные значения ограничиваются диапазоном `[0, 255]` и сохраняются как целые числа.

### Пример использования

```python
from filtr import filter_matrixSvertki

filter_matrixSvertki("input.jpg", "blurred.jpg", 6)
```

## Фильтр Кувахара

**Функция:** `filter_Kuwahara(source_name, result_name)`

Реализует фильтр Кувахара — нелинейный сглаживающий фильтр, сохраняющий резкие границы изображения при подавлении шумов.

### Параметры

| Параметр      | Тип   | Описание                            |
| ------------- | ----- | ----------------------------------- |
| `source_name` | `str` | Путь к входному изображению         |
| `result_name` | `str` | Имя файла для сохранения результата |

### Алгоритм

1. Изображение загружается в массив NumPy.
2. Создаётся копия изображения для сохранения результата.
3. Определяется внутренняя область пикселей, исключая по четыре пикселя с каждой стороны изображения.
4. Для каждого пикселя:

   * формируются четыре блока размером `3×3`;
   * вычисляются средние значения и дисперсии;
   * выбирается блок с минимальной дисперсией;
   * пикселю присваивается среднее значение выбранного блока.
5. Крайние пиксели остаются без изменений.
6. Результат сохраняется в файл.

Фильтр Кувахара эффективен для подавления шумов при сохранении чёткости границ и широко применяется при обработке медицинских и спутниковых изображений.

### Пример использования

```python
from filtr import filter_Kuwahara

filter_Kuwahara("input.jpg", "kuwahara_output.jpg")
```

## Фильтр Куана

**Функция:** `filter_Kuan(source_name, result_name, ch_vib)`

Реализует адаптивный фильтр Куана для подавления мультипликативного шума, например спекл-шума. Фильтр использует локальную статистику в окне размером `7×7` и глобальную дисперсию изображения для адаптивного сглаживания при сохранении резких границ и деталей.

### Параметры

| Параметр      | Тип     | Описание                                                              |
| ------------- | ------- | --------------------------------------------------------------------- |
| `source_name` | `str`   | Путь к входному изображению                                           |
| `result_name` | `str`   | Имя файла для сохранения результата                                   |
| `ch_vib`      | `float` | Параметр, характеризующий уровень шума и влияющий на коэффициент `CU` |

### Алгоритм

1. Изображение загружается в массив NumPy.
2. Для каждого цветового канала вычисляется глобальная дисперсия.
3. Для каждого пикселя внутренней области изображения (без четырёх крайних строк и столбцов):

   * извлекается окно размером `7×7`;
   * вычисляются локальное среднее `LM` и локальная дисперсия `LV`;
   * определяется значение текущего пикселя `PC`.
4. Вычисляются коэффициенты:

```text
CU = 1 / sqrt(ch_vib) * глобальная_дисперсия

CI = sqrt(LV) / LM * глобальная_дисперсия
```

5. Рассчитывается адаптивный коэффициент:

```text
K = (1 - (CU² / CI²)) / (1 + CU²)
```

6. Итоговое значение пикселя вычисляется по формуле:

```text
R = PC * K + LM * (1 - K)
```

7. Результат ограничивается диапазоном `[0, 255]`.
8. Краевые пиксели остаются без изменений.
9. Обработанное изображение сохраняется в файл.

Фильтр Куана эффективен для подавления спекл-шума в радиолокационных, ультразвуковых и медицинских изображениях, сохраняя при этом мелкие детали и границы объектов.

### Пример использования

```python
from filtr import filter_Kuan

filter_Kuan("input.jpg", "kuan_output.jpg", 0.5)
```

## Установка и использование

### Требования

* Python 3.7 или выше
* Библиотеки:

  * `numpy >= 1.20`
  * `Pillow`

### Установка зависимостей

Создайте файл `requirements.txt`:

```text
numpy
Pillow
```

Затем выполните:

```bash
pip install -r requirements.txt
```
